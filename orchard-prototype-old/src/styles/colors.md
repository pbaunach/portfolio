# Orchard Color Palette

This document defines the official color palette for the Orchard Career Counselor app.

## Brand Colors

### Primary Brand
- **Orchard Logo Blue**: `#13c7e5` - Primary brand color, used for main actions and highlights
- **Orchard Logo Green**: `#3CD343` - Secondary brand color, used for success states and positive actions

## Midnight Color Palette

### Dark Theme Base
- **Midnight Dark**: `#08121D` - Main background color
- **Midnight Medium**: `#0C2235` - Card backgrounds and secondary surfaces
- **Midnight Light**: `#223B4C` - Borders and dividers
- **Midnight Lighter**: `#516F84` - Subtle text and inactive elements

## Neutral Colors

### Text and UI Elements
- **Slate**: `#B5BFC8` - Primary text color
- **Slate Light**: `#E4E9ED` - Secondary text and labels
- **Cauliflower**: `#FFFFFF` - Pure white for high contrast text and backgrounds

## Accent Colors

### Vibrant Accents
- **Dragon Fruit**: `#F759B3` - Pink accent for special features
- **Orange**: `#F29820` - Orange accent for warnings and highlights
- **Plum**: `#BE13E8` - Purple accent for premium features
- **Lemon**: `#E8F948` - Yellow accent for attention-grabbing elements
- **Raspberry**: `#EA1B66` - Red accent for errors and urgent actions
- **Blueberry**: `#3372FF` - Blue accent for links and secondary actions

## Usage Guidelines

### Text Hierarchy
- **Primary Text**: `text-slate-light` (#E4E9ED)
- **Secondary Text**: `text-slate` (#B5BFC8)
- **High Contrast**: `text-cauliflower` (#FFFFFF)

### Background Hierarchy
- **Main Background**: `bg-midnight-dark` (#08121D)
- **Card Background**: `bg-midnight-medium` (#0C2235)
- **Border**: `border-midnight-light` (#223B4C)

### Interactive Elements
- **Primary Button**: `bg-orchard-blue text-cauliflower`
- **Secondary Button**: `bg-midnight-medium text-slate-light border-midnight-light`
- **Success Button**: `bg-orchard-green text-cauliflower`
- **Warning Button**: `bg-orange text-cauliflower`
- **Error Button**: `bg-raspberry text-cauliflower`

### Status Colors
- **Success**: `text-orchard-green` or `bg-orchard-green`
- **Warning**: `text-orange` or `bg-orange`
- **Error**: `text-raspberry` or `bg-raspberry`
- **Info**: `text-orchard-blue` or `bg-orchard-blue`

## Accessibility

All color combinations meet WCAG AA contrast requirements:
- Text on dark backgrounds uses light colors for sufficient contrast
- Interactive elements have clear hover and focus states
- Color is not the only means of conveying information
